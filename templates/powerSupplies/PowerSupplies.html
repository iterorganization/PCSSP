PURPOSE: Simulates the input-output dynamics of the power supplies that feed all the axisymmetric coils/circuit, i.e. CS, PF, VS1 and VS3.
The coils are sorted as CS3U CS2U CS1 CS2L CS3L P1 P2 P3 P4 P5 P6.


CONFIGURATION INPUT (config struct):

nMConverters = Number of converters for the CS/PF power supplies
VMCpositiveSR = CS/PF power supplies postive slew rates (V/s)
VMCnegativeSR = CS/PF power supplies negative slew rates (V/s)
VMCtaud = CS/PF power supplies pure delay (s)
currentLim = CS/PF power supplies current limits (A)
voltageLim = CS/PF power supplies (symmetric) voltage limits (V)
nVS1Converters = Number of converters for the VS1 power supply
VS1positiveSR = VS1 power supply postive slew rate (V/s)
VS1negativeSR = VS1 power supply negative slew rate (V/s)
VS1taud = VS1 power supply pure delay (s)
VS3filterNum = Numerator of the VS3 power supply dynamical model
VS3filterDen = 'Denominator of the VS3 power supply dynamical model
VS3taud = VS3 power supply pure delay (s)
VS3Vup = VS3 upper voltage limit (V)
VS3Vlow = VS3 lower voltage limit (V)


INITIALIZATION INPUT (init struct):


INPUT SIGNALS:

CPSF_volt_cmd = commands to the CS/PF power supplies (V)
VS1_volt_cmd = command to the VS1 power supply (V)
VS3_volt_cmd = command to the VS3 power supply (V)
CSPF_curr_meas = currents in yhe CS/PF coils (A)


OUTPUT SIGNALS:

VCSPF = voltages applied to the CS/PF coils (V)
VS1 = voltage applied to the VS1 circuit (V)
VS3 = voltage applied to the VS3 circuit (V)


CODE GENERATION COMPATIBLE: yes/no

YES


RESTRICTIONS:

None.
